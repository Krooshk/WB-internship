##### В данной папке находится проект "Генератор мемов".
---
> 2 Генератор мемов  [code sourse of generator-memes](https://github.com/Krooshk/WB-internship/tree/main/L2/generator-memes) |
[deploy generator-memes](https://krooshk.github.io/WB-internship/L2/generator-memes/src/)

 Используемые технологии:
 - HTML
 - CSS
 - JS

Было реализовано поле куда можно добавить картинки для дальнейшего редактирования. Доступно два способа: с помощью Drag and Drop, и просто выбора через проводник. Начальная страница представлена на картинке ниже.

![generator1](https://github.com/Krooshk/WB-internship/blob/main/L2/generator-memes/src/assets/img/imageForReadme/generator1.jpg?raw=true)

Есть возможность добавления блоков с текстом, а также удаления их. Изменения размера,стиля шрифта, цвета границы букв и изменения заливки букв. Полученный мем можно сохранить локально на компьютере.

![generator2](https://github.com/Krooshk/WB-internship/blob/main/L2/generator-memes/src/assets/img/imageForReadme/generator2.jpg?raw=true)
