##### В данной папке находится проект "Визуализация алгоритмов"
---
8 Визуализация алгоритмов сортировки [code sourse of visualization](https://github.com/Krooshk/WB-internship/tree/main/L3/visualization) | [deploy visualization](https://krooshk.github.io/WB-internship/L3/visualization/src/)

 Используемые технологии:
 - HTML
 - CSS
 - JS

В данном проекте реализована визуализация алгоритмов сортировки
- Пузырьковая сортировка
- Сортировка выбором
- Сортировка вставкой
- Быстрая сортировка
- Сортировка кучей

Есть возможность изменить размер массива, а также изменить скорость анимации.
Во время выполнения алгоритма добавлена возможность остановки / воспроизведения алгоритма.
