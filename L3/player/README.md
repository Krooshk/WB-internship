##### В данной папке находится проект "Плеер".
---

- 5 Музыкальный плеер с визуализацией [code sourse of player](https://github.com/Krooshk/WB-internship/tree/main/L3/player) |
[deploy player](https://krooshk.github.io/WB-internship/L3/player/src/)


 Используемые технологии:
 - HTML
 - CSS
 - JS
 - wavesurfer.js

Был реализован плеер с возможность добавления аудизаписей с компьютера. 
    Основной функционал:
- Выбора трека из плейлиста
- Остановить и запустить воспроизведение можно как главной кнопкой, так и кликом по треку из плейлиста
- Отображение в плейлисте текущего трека
- Изменение громкости в плеере
- Визуализация волн с помощью wavesurfer
- Переключение на следующий и предыдущий трек
- Перемотка воспроизведения
- Отображение времени воспроизведения / длительности трека

Дополнительный функционал:
- Перемешивание треков
- Повторение трека
- Добавление аудизаписей с компьютера
