##### В данной папке находится проект "Многопользовательская игра крестики-нолики"
---
10 Многопользовательская игра на одном компьютере [code sourse of tic-tac](https://github.com/Krooshk/WB-internship/tree/main/L3/tic-tac) | [deploy tic-tac](https://krooshk.github.io/WB-internship/L3/tic-tac/src/)

 Используемые технологии:
 - HTML
 - CSS
 - JS

В данном проекте реализована популярная игра: крестики-нолики.
На главной странице есть возмодность выбора режима игры: с другом, либо с компьютером.
Логотип в левом верхнем углу является кнопкой  "home", он перенаправляет на главную страницу.

![mainpage](https://github.com/Krooshk/WB-internship/blob/main/L3/tic-tac/src/assets/img/imageForReadme/tictac.jpg?raw=true)

--- 

В режиме игры с другом следует ходить поочередно. В случае победы одного из участников будет подсвечена выигрышная комбинация. В случае ничьей клетки окрасятся в зеленый цвет.

![tictacfriend](https://github.com/Krooshk/WB-internship/blob/main/L3/tic-tac/src/assets/img/imageForReadme/tictacfriend.jpg?raw=true)

---

В режиме игры с компьютером есть два режима - легкий и сложный. В легком режиме компьютер выбирает следующий ход наугад, в сложном режиме компьютером используется алгоритм минимакс.
Для переключения режима используйте переключатель вверху справа.

![tictacComp](https://github.com/Krooshk/WB-internship/blob/main/L3/tic-tac/src/assets/img/imageForReadme/tictacComp.jpg?raw=true)
